# Command Reference Documentation

You are documenting the complete command reference for grove-meta, the meta-CLI and package manager for the Grove ecosystem.

## Task
Write a comprehensive command reference that:
- Documents all available commands and subcommands
- Provides detailed usage information for each command
- Includes all flags, options, and arguments
- Shows practical examples for common use cases
- Explains command output and return values

## Commands to Document
Analyze the codebase to identify all commands including:
- Core grove commands (list, status, install, etc.)
- Workspace management commands
- Tool management commands
- Configuration commands
- Utility and helper commands

## Format for Each Command
For each command, provide:
- **Synopsis**: Brief one-line description
- **Syntax**: Complete command syntax with options
- **Description**: Detailed explanation of what the command does
- **Options/Flags**: All available flags with descriptions
- **Arguments**: Required and optional arguments
- **Examples**: Multiple practical examples
- **Output**: Description of command output
- **Exit Codes**: Return values and their meanings
- **Related Commands**: Cross-references to similar commands

## Special Considerations
- Document both short and long flag formats
- Explain environment variable interactions
- Note any command aliases
- Include global flags that apply to all commands
- Document interactive prompts and inputs
- Explain configuration precedence

## Output Format
Create a well-structured Markdown document organized alphabetically or by functional groups, with consistent formatting, clear examples, and easy navigation between commands.
