# Getting Started Documentation

You are documenting the getting started guide for grove-meta, the meta-CLI and package manager for the Grove ecosystem.

## Task
Write a beginner-friendly getting started guide that:
- Walks new users through their first grove-meta experience
- Demonstrates core functionality with practical examples
- Builds confidence through achievable quick wins
- Establishes best practices from the beginning
- Provides a clear learning path to more advanced features

## Key Topics to Cover
- First-time setup and initialization
- Understanding workspaces and projects
- Basic grove commands and their usage
- Managing Grove ecosystem tools
- Creating your first Grove-managed project
- Common workflows and patterns

## Practical Examples to Include
- Listing available Grove tools with `grove list`
- Installing and managing Grove ecosystem tools
- Creating and navigating workspaces
- Building projects with make
- Running tests with tend
- Managing binaries in ./bin

## Progressive Learning Path
- Start with simple commands (list, status)
- Move to workspace management
- Introduce tool installation and updates
- Show integration with other Grove tools
- Demonstrate development workflows

## Best Practices to Establish
- Keeping tools updated
- Organizing workspaces effectively
- Using make targets appropriately
- Following Grove conventions
- Leveraging the ecosystem effectively

## Output Format
Create a well-structured Markdown document with step-by-step tutorials, clear command examples, expected outputs, and helpful tips for new users.
